# ruby_spec.nvim
* rspec commands!

## Installation
* Plug

```
Plug 'senkentarou/ruby_spec.nvim'
```

## Usage
* Open toggle rspec file.
  * `:ToggleRspecFile`
* Run all rspec on file.
  * `:RunRspec`
* Run rspec at line.
  * `:RunRspecAtLine`

## For development
* Load under development plugin files on root repository.

```
nvim --cmd "set rtp+=."
```
